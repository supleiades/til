置換(ノーマルモード --> : )

```
範囲内のマッチする文字列を置換
:'<,'>s///g
```

s：置換
%s：ファイル内の行置換
g：行内の複数すべて置換
c：確認


```
マッチした列を削除
:g//d
```

```
マッチした列以外を削除
:g//v
```

改行を削除
```
:s/\n//g
```
各行を挿入
```
:s//\r/g
```
（注意）挿入と削除で特殊文字が変わる

検索して、ハイライトした文字列を置換
```
/<置換前の文字列>
:%s//<置換後の文字列>/g
```

## 確認しながら置換
```sh
:%s/<置換前の文字列>/<置換後の文字列>/gc

# replace with hoge (y/n/a/q/l/^E/^Y)?
# y:  Yes
# n:  No
# a:  All
# q:  Quit
# l:  Line カレント行にだけ適用する
# ^E: Ctrl+e で上にスクロール
# ^Y: Ctrl+y で下にスクロール
```
