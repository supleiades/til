- タグはコミットに付与される
- タグは一般的なcommit/push( git push origin main )では一緒に反映されない
- 別途コマンドを打つ必要がある

### 登録したタグを一括でリモートに反映させる方法
`git push origin --tags`

### タグにコメントを付与する
`git tag -a {タグ名} -m '{タグのコメント}'`
